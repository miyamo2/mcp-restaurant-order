# mcp-restaurant-order

MCP server implementation. Just a toy.

## Quick Start

### Install

```sh
go install github.com/miyamo2/mcp-restaurant-order@latest
```

### Use in Claude Desktop

```json
{
    "mcpServers": {
        "mcp-restaurant-order": {
          "command": "mcp-restaurant-order"
        }
    }
}
```
